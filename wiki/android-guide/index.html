Android

지원환경

  • 개발환경 : Android Studio 사용을 권장합니다.(Eclipse는 미지원)
  • 지원버전 : API level 21 이상 지원(Lollipop)
  • AdFit SDK v3.11.10 부터 Android 12 (타겟 API 레벨 31 이상)을 대응합니다.

지원 광고유형

  • 배너 타입 광고
    • 지원 사이즈
      • 300 x 250
      • 320 x 100
      • 320 x 50
  • 네이티브 타입 광고
    • 지원 유형
      • 1200 * 600 메인 이미지 피드

앱 등록시 타겟층 설정

  • AdFit을 통해서 광고를 노출하는 경우, 안드로이드앱을 Google Play Console에 제출할 때 앱의 대상 연령대를 반드시 설정해야 합니다.
  • 앱의 대상 연령대가 일반 사용자(만 13세 이상)인 경우 가족 정책이 적용되지 않으나, 어린이(만 13세 미만)를 대상 연령대에 포함할 경우 Google의 가족 정책을 준수해야 합니다.
  • 앱의 대상 연령대에 어린이(만 13세 미만)가 포함될 경우, 광고 정책 설정 시 Google Play의 가족용 광고 프로그램 또는 앱의 중립적인 연령 심사 구현 여부에 관한 질문이 표시될 수도 있습니다.

개발 가이드

AdFit SDK를 다운로드하여 매체에 설치 후 광고를 설정합니다.
최신 AdFit SDK 다운로드 및 상세 개발 가이드는 AdFit Android SDK Github을 통해 확인 가능합니다.

이 문서는 Kakao 신디케이션 제휴 당사자에 한해 제공되는 자료로 가이드 라인을 포함한 모든 자료의 지적재산권은 주식회사 카카오가 보유합니다.
