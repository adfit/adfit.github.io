개발 가이드

AdFit에서 배너 유형의 광고단위를 생성 시 아래와 같은 형식의 광고 설정 스크립트가 생성됩니다.

광고를 노출할 페이지에 AdFit에서 생성된 광고 설정 스크립트를 삽입합니다.
광고 스크립트 내 광고단위 정보를 변경하거나 코드를 수정해서 설치할 경우, 광고 요청에 실패하거나 잘못된 광고 요청으로 처리될 수 있습니다.

상세 개발 가이드는 AdFit Web SDK Github을 통해 확인 가능합니다.
